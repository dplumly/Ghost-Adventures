Ghost-Adventures
================

Arduino Ghost Detecting Device


Supplies:

LED
PIR Sensor

Arduino Yun (if you want to send a text to a device) If not use any other arduino eg. Uno

Jumpers

3D printed ghost

Base (I got the top of a fence post from Lowes, but any base will do)
